# 科技主题轨迹数据库管理系统

**面向精准产业升级的科技主题轨迹数据库管理系统研究**

> 湖北省重点研发计划项目（2023BAB081）｜武汉大学牵头实施

**在线平台：[http://db4s.swangdbs.com/](http://db4s.swangdbs.com/)**

---

## 项目简介

本系统是一套面向科技情报分析与精准产业升级的**科技主题轨迹数据库管理系统**，融合知识图谱、多模态大模型与高维向量检索技术，实现对海量科技文献、专利数据的主题轨迹建模、演化分析与智能情报服务。

### 核心目标

- 攻克海量异构科技数据的解析与演化建模难题
- 构建基于知识图谱与多模态大数据的科技主题轨迹数据库
- 打通底层数据中枢与前台应用场景，支撑产研协同
- 提供主题追踪、智能查询、产研合作推荐、区域生态分析四位一体的情报服务能力

## 系统架构

系统采用"数据—模型—平台—应用"四层架构：

```
┌──────────────────────────────────────────────────────┐
│  应用层  │ 轨迹对比 │ 论文轨迹 │ 专利轨迹 │ 地图可视化 │
│          │ 趋势预测 │ 区域创新 │ 合作推荐 │ AI报告生成 │
├──────────────────────────────────────────────────────┤
│  服务层  │  大语言模型 (DeepSeek/Qwen)                │
│          │  MCP 协议网关（调度·并发分发·解耦）        │
│          │  多模态 RAG（向量/图/结构化三源融合）       │
├──────────────────────────────────────────────────────┤
│  数据层  │ 科技主题轨迹库 │ 主题对象库 │ 语义向量索引 │
│          │         主题关系知识图谱（10亿+三元组）     │
├──────────────────────────────────────────────────────┤
│  数据接入 │ 数据源→采集→清洗→信息抽取→向量化→图构建→入库 │
└──────────────────────────────────────────────────────┘
```

## 关键技术

### 1. 科技主题轨迹高维表征与多尺度建模

对论文作者、专利发明人、论文机构、专利申请人等异构对象进行统一建模，采用 **384 维语义嵌入向量**统一表征主题轨迹对象，支持人才画像、团队协作、机构态势、领域演化等多尺度分析。

### 2. 高效轨迹相似性计算 BioVSS

通过生物启发式稀疏编码与候选集剪枝，在保持 **93.8%–98.9% 召回率**的基础上实现 **46–78 倍加速**，突破线性扫描在海量轨迹数据上的效率瓶颈。

| 方法 | CS 耗时 | 加速比 | 召回率 |
|------|---------|--------|--------|
| Brute Force | 9.16s | 1× | 100% |
| BioVSS | 0.73s | 12× | 97% |
| BioVSS++ | **0.20s** | **46×** | 98% |

### 3. 知识图谱驱动的多模态智能查询（MM-CoT）

以自建科技主题轨迹数据库为统一数据底座，由 MM-CoT 控制器协调**向量、图与关系**三种模态，支持意图识别、任务分解、模态调度与结果融合，生成可溯源答案。
